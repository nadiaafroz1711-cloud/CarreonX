# CarreonX — AI-Powered Career Intelligence Platform 🚀

**CarreonX** is a premium, production-ready career guidance system that bridges the gap between learning and professional mastery. Using Google Gemini Pro AI, it generates personalised roadmaps, identifies skill gaps, and provides 24/7 AI mentorship.

---

## ✨ Key Features
- **🗺️ AI Career Roadmap**: Instant, step-by-step learning paths tailored to your goal.
- **🤖 AI Mentor Chat**: 24/7 technical and career coaching with an LLM-powered assistant.
- **📊 Learning Analytics**: Animated dashboard to track your skills and test progress.
- **🧪 Mock Tests**: Subject-specific quizzes generated by AI for self-assessment.
- **📺 Integrated Resources**: Curated YouTube tutorials for every skill milestone.

---

## 🛠️ Technology Stack
### Frontend
- **Next.js 16 (React)**: High-performance App Router system.
- **TypeScript**: Ensuring reliable, type-safe development.
- **Vanilla CSS**: Custom "Glassmorphism" UI for a state-of-the-art feel.

### Backend
- **FastAPI (Python)**: Modern, asynchronous API framework.
- **SQLAlchemy (ORM)**: Seamless database management.
- **PostgreSQL**: Robust data persistence for production.

### AI Core
- **Google Gemini Pro 1.5**: Advanced LLM for reasoning and content generation.

## 🚀 Getting Started (Local Development)

### 1. Backend Setup
```bash
cd backend
python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate
pip install -r requirements.txt
# Create a .env file with your GEMINI_API_KEY
uvicorn main:app --reload
```

### 2. Frontend Setup
```bash
cd careerai-web
npm install
# Create a .env.local with NEXT_PUBLIC_API_URL=http://localhost:8000
npm run dev
```
